## Getting Started


### Prerequisites

Install node
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bittersoftware/my-personal-page-svelte.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your Githu API in `.env`
   ```sh
   PRIVATE_GITHUB_API=YOUR_API
   ```
4. Run the server
   ```sh
   npm run dev
   ```
